# AlipayARHB

# How to Use

#### 1、restore one picture
🌰  python alipayar.py input/IMG_1838.PNG
#### 2、restore a lot of pictures
🌰  python alipayar.py input/ output/

run:
./hb.sh

# Requirement
Python Imaging Library(PIL)

you can use pip to install:
sudo pip install pillow
